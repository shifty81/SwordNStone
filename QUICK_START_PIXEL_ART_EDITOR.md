# Quick Start: Pixel Art Skin Editor

## 5-Minute Tutorial

### Step 1: Open the Editor
1. Main Menu ‚Üí **Singleplayer**
2. Click **"Character..."**
3. Click **"Skin Editor"** (bottom-right)

### Step 2: Load a Template
1. Click **"Load Template"** button
2. You now have a basic character with default underwear
3. This is your starting canvas!

### Step 3: Choose Your Tool
Click one of these tool buttons:
- **Brush** - Draw pixels
- **Eraser** - Remove pixels
- **Fill** - Fill areas with color
- **Picker** - Sample colors

### Step 4: Pick a Color
Use the RGB sliders:
- **R (Red):** Click **+** or **-** to adjust (0-255)
- **G (Green):** Click **+** or **-** to adjust (0-255)
- **B (Blue):** Click **+** or **-** to adjust (0-255)

**Example Colors:**
- Black: R=0, G=0, B=0
- White: R=255, G=255, B=255
- Blue: R=0, G=0, B=255
- Skin: R=220, G=180, B=140

### Step 5: Start Drawing
1. Make sure **Brush** tool is selected
2. Click and drag on the canvas to draw
3. Watch the 3D preview update in real-time!

### Step 6: Create Your Design

**For a Simple T-Shirt:**
1. Select a shirt color (e.g., R=80, G=140, B=180 for blue)
2. Use **Brush** to paint the upper body area
3. Or use **Fill Bucket** to fill larger areas quickly

**For Hair:**
1. Click **"Overlay"** button to switch layers
2. Select hair color (e.g., R=80, G=50, B=20 for brown)
3. Paint hair on the head region
4. Hair will appear on top of the base layer

### Step 7: Fix Mistakes
1. Click **Eraser** tool
2. Click/drag to remove unwanted pixels
3. Or use **Color Picker** to match colors
4. Use **Brush** to paint over mistakes

### Step 8: Save Your Creation
1. Click **"Save Skin"** button (bottom-right)
2. Your skin is saved as a PNG file
3. It's automatically loaded in-game!

## Common Tasks

### Change Brush Size
- Click **-** button to make smaller (minimum 1)
- Click **+** button to make larger (maximum 10)
- Use size 1 for details, size 5 for filling

### Fill Large Areas
1. Select **Fill Bucket** tool
2. Choose your color
3. Click on the area you want to fill
4. All connected pixels of the same color will change

### Copy a Color
1. Select **Color Picker** tool
2. Click on any pixel in the canvas
3. That color becomes your selected color
4. Now draw with that exact color!

### Switch Layers
- **Base Layer** - Body, clothes (click "Base Layer")
- **Overlay** - Hair, accessories (click "Overlay")
- Each layer is independent

### Preview Your Work
- Look at the **3D Preview** panel on the right
- It updates automatically as you draw
- Switch **Male/Female** to see both versions

### Start Over
1. Click **"Load Template"** to reload default
2. Or use **Eraser** on large areas
3. Or click **"Clear"** to wipe everything

## Pro Tips

‚úÖ **DO:**
- Start with Base Layer for body/clothes

- Use small brush (1-2) for details
